# Contributing to "The New Guy"

First off, thank you for considering contributing to **A Dev's Journey: The New Guy**! This is an open-source educational game aimed at making learning full-stack development engaging, stressful (in a fun way), and highly interactive. 

Because we are blending game design, web development, and local AI models, there is a lot of ground to cover. We welcome developers, designers, and educators of all skill levels.

## 🎯 What We Need Help With Right Now

Since we are in the early stages (Phase 1), we are actively looking for help with:
1. **React / Tailwind Wizards:** Helping design the 8-bit, neon-themed IDE interface and menu screens.
2. **Electron Setup:** Structuring the boilerplate to handle local file system operations (for the code editor) securely.
3. **Prompt Engineers / AI Enthusiasts:** Helping write the system prompts and personality constraints for our AI NPCs (The PM, Tech Lead, and QA).
4. **Pixel Artists:** Creating 2D, minimal 8-bit avatars for the character creation screen.

## 🛠️ How to Contribute

1. **Fork the Repository:** Create your own copy of the project.
2. **Create a Branch:** Create a feature branch (`git checkout -b feature/neon-buttons` or `fix/typo-in-readme`).
3. **Commit your Changes:** Make your changes and commit them with a clear, descriptive message.
4. **Push to the Branch:** Push your changes to your fork (`git push origin feature/neon-buttons`).
5. **Open a Pull Request (PR):** Submit a PR against our `main` branch. 

## 🤖 A Note on the AI Models

This game relies on **local LLMs** (via Ollama) to function. We are strictly avoiding cloud-based APIs (like OpenAI) for the core gameplay to ensure the game remains free, private, and capable of running offline. If you are contributing to the AI logic, please test your integrations against lightweight local models (e.g., Llama 3 8B or similar).

## 🧑‍💻 Code of Conduct

Be respectful, be collaborative, and remember that this project is designed to help people learn. We treat our contributors with the same patience our "Tech Lead" NPC treats the player!
